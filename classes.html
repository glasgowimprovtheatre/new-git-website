Refund Policy

- Full refunds are available if requested by email 24 hours before the start of the course

- If the course has started, no refunds are available after the 1st class

- A partial refund of the remaining classes is offered if a refund is requested before the 2nd class

- In all cases we must receive a written request for a refund via email (glasgowimprovtheatre@gmail.com)

- In certain cases we will provide refunds due to exceptional circumstances
